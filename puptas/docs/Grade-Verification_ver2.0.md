|    |                                                                                                                                            |                                                          |            |
|:---|:-------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------|:-----------|
|    |                                                                                                                                            |                                                          |            |
|    | How to use this calculator:                                                                                                                |                                                          |            |
|    |                                                                                                                                            |                                                          |            |
|    | 1. Check the SHS Strand of the applicant on the Route and Approval Slip and click:                                                         |                                                          |            |
|    |                                                                                                                                            |                                                          |            |
|    |                                                                                                                                            | Strands                                                  | Links      |
|    |                                                                                                                                            | Accountancy, Business and Management (ABM)               | Click here |
|    |                                                                                                                                            | Science, Technology, Engineering, and Mathematics (STEM) | Click here |
|    |                                                                                                                                            | Humanities and Social Sciences (HUMSS)                   | Click here |
|    |                                                                                                                                            | General Academic Strand (GAS)                            | Click here |
|    |                                                                                                                                            | Information Communications Technology (ICT)              | Click here |
|    |                                                                                                                                            | Technical-Vocational (Tech-Voc)                          | Click here |
|    |                                                                                                                                            |                                                          |            |
|    | 2. Encode the Grade 11 and Grade 12 English, Math, and Science 2ubjects.                                                                   |                                                          |            |
|    |                                                                                                                                            |                                                          |            |
|    | 3. Encode the Grade 12 First Sem and Second Sem Average to compute the GWA.                                                                |                                                          |            |
|    |                                                                                                                                            |                                                          |            |
|    | 4. The calculator automatically recommends the programs where the applicant qualifies.                                                     |                                                          |            |
|    |                                                                                                                                            |                                                          |            |
|    | NOTE: ABM and HUMSS graduates cannot apply for BSIT (highlighted) so even if their grades qualify for BSIT, their strands are not aligned. |                                                          |            |
|    |                                                                                                                                            |                                                          |            |
|    | 5. Write the grades and check the programs on the slip and give the slip to the applicant before asking them to log on the log sheet.      |                                                          |            |
